# Godot Web Build Compressor
## Description

A tool for automatic compression of a Godot game build for the web platform. The utility optimizes file sizes and modifies code to work with compressed resources.

## Main features

- **File compression**: .wasm and .pck files are compressed with a selectable compression level
- **Automatic JavaScript modification**: Replaces functions to support compressed resources
- **pako integration**: Adds the library for client-side decompression
- **Flexible settings**: Choose the compression level for each file type
- **Backups**: Optional creation of backups before modifying files
## Описание

Программа для автоматического сжатия билда игры на Godot для веб-платформы. Утилита оптимизирует размер файлов и модифицирует код для работы со сжатыми ресурсами.

## Основные функции

- **Сжатие файлов**: WASM и PCK файлы сжимаются с выбранным уровнем компрессии
- **Автоматическая модификация JavaScript**: Замена функций для поддержки сжатых ресурсов
- **Интеграция pako**: Добавление библиотеки для распаковки на клиентской стороне
- **Гибкие настройки**: Выбор уровня сжатия для каждого типа файлов
- **Резервные копии**: Опциональное создание бэкапов перед изменением
